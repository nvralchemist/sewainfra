# Sewain Checkout Prototype

Single-page Next.js prototype for a WhatsApp-style rental booking checkout flow.

## What it includes

- Mobile-first checkout page
- Fake booking data
- Fake payment method selection
- Countdown timer for payment hold
- No backend
- No login

## Tech stack

- Next.js
- React
- Tailwind CSS
- TypeScript

## Local development

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Deploy to Vercel

1. Push this project to a Git repository.
2. Import the repository into Vercel.
3. Deploy with the default Next.js settings.

No environment variables are required for this prototype.
